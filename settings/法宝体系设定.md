# 法宝体系设定

## 一、法宝等级体系

### 1.1 等级划分
| 等级 | 名称 | 对应修为 | 特点 | 示例 |
|------|------|----------|------|------|
| **一阶** | 法器 | 炼气期 | 基础功能，无灵智 | 普通飞剑、低阶符箓 |
| **二阶** | 灵器 | 筑基期 | 初步灵智，可认主 | 中品飞剑、储物袋 |
| **三阶** | 法宝 | 金丹期 | 强大威能，器灵雏形 | 本命法宝、阵旗 |
| **四阶** | 灵宝 | 元婴期 | 完整器灵，神通自生 | 镇宗灵宝、古宝 |
| **五阶** | 古宝 | 化神期 | 上古传承，规则碎片 | 传承古宝、通天灵宝 |
| **六阶** | 通天灵宝 | 炼虚期+ | 蕴含天道规则 | 极少现世，传说级 |

### 1.2 金融相关特殊分类
| 类型 | 等级 | 功能 | 示例 |
|------|------|------|------|
| **契约类** | 法器-灵宝 | 签订、存储、验证契约 | 玉简、灵契印、公证印 |
| **计算类** | 法器-法宝 | 金融计算、数据分析 | 算筹、灵纹阵盘 |
| **通讯类** | 法器-灵器 | 信息传递、市场通讯 | 传讯符、灵讯台 |
| **存储类** | 法器-法宝 | 财富存储、资产保管 | 储物袋、灵萃匣、宝库 |

## 三、使用规范

### 3.1 单位规范（与单位体系一致）
| 物品类型 | 单位 | 示例 |
|----------|------|------|
| 飞剑类 | 把、柄 | 一把青锋剑、三柄飞剑 |
| 法宝类 | 件、个 | 一件护身法宝、三个攻击法宝 |
| 符箓类 | 张、道 | 一张火球符、三道雷符 |
| 玉简类 | 枚、片 | 一枚空白玉简、三片传讯玉简 |
| 阵旗类 | 面、套 | 一面阵旗、一套五行阵旗 |

### 3.2 功能描述规范
- **攻击类**：描述威力、速度、特殊效果
- **防御类**：描述防护范围、强度、持续时间
- **辅助类**：描述辅助效果、消耗、限制
- **特殊类**：描述独特功能、使用条件

### 3.3 认主与使用
1. **认主方式**：滴血认主、神识烙印、灵力温养
2. **使用消耗**：灵力、神识、特殊材料
3. **限制条件**：修为要求、属性匹配、使用次数
4. **成长性**：部分法宝可随主人成长

## 六、伏笔与成长

### 6.1 陆衍法宝成长线
1. **炼气期**：基础金融工具（玉简、计量阵）
2. **筑基期**：风险工具（评估盘、监控阵）
3. **金丹期**：规则工具（契约重器、市场核心）
4. **元婴期**：天道工具（规则修改、新秩序）

### 6.2 重要伏笔
1. **青蚨钱纹**：镇库灵宝已遭私炼，蕴含危机
2. **眼角刻痕**：连接两界金融，可能引动共振
3. **灵契公证印**：九宫格空白，预留反制手段
4. **天机阁监察**：可能有更高阶法宝监控陆衍

### 6.3 法宝冲突设计
1. **传统vs创新**：古宝与金融新法宝的碰撞
2. **规则vs突破**：现有法宝规则限制金融创新
3. **个人vs宗门**：个人法宝挑战宗门传承法宝

## 七、创作注意事项

### 7.1 一致性检查
- [ ] 新出现法宝是否符合等级体系
- [ ] 单位使用是否规范（把/件/张）
- [ ] 功能描述是否与等级匹配
- [ ] 使用限制是否合理

### 7.2 创新性要求
- [ ] 金融法宝要有独特设计理念
- [ ] 传统法宝要有修仙特色
- [ ] 避免现代物品简单修仙化
- [ ] 保持修仙世界的韵味

### 7.3 平衡性维护
- [ ] 法宝威力与修为匹配
- [ ] 金融法宝不过于逆天
- [ ] 传统法宝仍有价值
- [ ] 成长有合理节奏

## 八、示例写法

### 正确示例
```
陆衍取出一枚空白玉简（法器），注入灵识。
玉简表面浮现淡金灵纹，形成标准化契约模板。
这是他用金融之眼设计的"灵契玉简"，虽只是法器，
但契约结构之严谨，堪比灵器级公证印。
```

### 错误示例
```
陆衍拿出一个平板电脑（❌现代物品直接出现）
上面显示着K线图（❌未修仙化）
他用手指滑动操作（❌未说明灵力驱动）
```
