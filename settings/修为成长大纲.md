# 修为成长大纲

## 一、修仙等级体系

### 1.1 标准等级划分
```
炼气期（1-12层） → 筑基期（初/中/后） → 金丹期（初/中/后） → 元婴期（初/中/后） → 化神期
```

### 1.2 各阶段特点
| 阶段 | 寿命 | 能力特点 | 金融对应 |
|------|------|----------|----------|
| **炼气期** | 100-120年 | 基础灵力运用，法器使用 | 个人金融操作 |
| **筑基期** | 200-250年 | 灵力液化，灵器认主 | 坊市级金融体系 |
| **金丹期** | 500-800年 | 灵力固化，本命法宝 | 跨城金融网络 |
| **元婴期** | 1000-1500年 | 元婴出窍，规则感悟 | 洲际金融规则 |
| **化神期** | 2000-3000年 | 神通自生，触及天道 | 天道金融秩序 |

## 五、心魔与风险体系

### 5.1 金融之眼特有风险
```
心魔类型：量化自我残留
表现：多个"过去的自己"报出不同估值
风险：决策干扰，过度理性，丧失修仙感性
应对：清心丹辅助，平衡修炼，情感体验
```

### 5.2 修为提升风险
| 阶段 | 主要风险 | 金融对应风险 | 解决方案 |
|------|----------|--------------|----------|
| **筑基** | 心魔反噬 | 信用体系崩溃 | 风险分散，备用方案 |
| **金丹** | 天劫降临 | 系统性风险 | 风险对冲，规则保护 |
| **元婴** | 天道排斥 | 规则冲突 | 渐进改革，天道适应 |

### 5.3 幽默风险场景
```
散修筑基时：心魔是"担心信用分下降"
钱掌柜突破：心魔是"算错账的噩梦"

```

### 6.1 时间线控制
```
第一章：穿越当天
第二章：第3-10天（存单推出）
第三章：第11-30天（信用体系）
第四章：第31-60天（二级市场）

第六章：第91-120天（规则挑战）
第七章：第121-180天（帝国雏形）
```

### 6.2 修为提升合理性
- 陆衍：金融之眼辅助，资源充足，合理快速
- 钱掌柜：传统修炼，速度正常
- 散修们：资源有限，速度较慢
- 沈砚：天赋+资源，速度中等偏快

### 6.3 资源获取逻辑
```
陆衍：金融收益 → 购买丹药/法器 → 修为提升
散修：参与金融 → 获得收益 → 改善修炼
钱庄：传统收益 → 保守投资 → 缓慢提升
宗门：固有资源 → 内部分配 → 稳定提升
```

## 七、创作规范

### 7.1 修为描述规范
```
正确：陆衍运转功法，灵力在经脉中流转三周天
错误：陆衍等级提升到15级（❌游戏化）
正确：筑基成功，灵力液化，神识范围扩大
错误：获得新技能"金融风暴"（❌技能化）
```

### 7.2 丹药使用规范
```
正确：服下一枚培元丹，药力化开温养经脉
错误：吃下经验丹升了一级（❌游戏化）
正确：筑基丹入口即化，冲击瓶颈
错误：使用道具"突破符"（❌道具化）
```

### 7.3 法器使用规范
```
正确：祭出风险评估盘，灵光扫描契约
错误：装备"金融之眼"（❌装备化）
正确：玉简认主，神识可随时查阅
错误：绑定物品"契约书"（❌绑定化）
```
