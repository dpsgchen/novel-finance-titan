# 修仙界货币体系设定

## 一、货币单位体系

### 1.1 基础单位（三级制）
```
【灵萃】 - 最小单位，类似“分”
  符号：ⓛ（可简写为“萃”）
  用途：日常小额交易，最低端物品

【灵珠】 - 中等单位，类似“角”
  符号：ⓖ（可简写为“珠”）
  换算：1灵珠 = 100灵萃
  用途：中等交易，常见物品

【灵石】 - 标准单位，类似“元”
  符号：ⓢ（可简写为“石”）
  换算：1灵石 = 10灵珠 = 1000灵萃
  用途：大额交易，重要物品
```

### 1.2 纯度体系（四级制）
```
【下品灵石】 - 标准纯度，流通最广
  灵气含量：标准单位
  颜色：淡白色
  流通性：极高

【中品灵石】 - 高纯度
  换算：1中品灵石 = 100下品灵石
  灵气含量：下品灵石的100倍
  颜色：乳白色
  用途：高阶修士交易，大型交易

【上品灵石】 - 极高纯度
  换算：1上品灵石 = 100中品灵石 = 10,000下品灵石
  灵气含量：下品灵石的10,000倍
  颜色：淡蓝色
  用途：元婴以上修士，宗门级交易

【极品灵石】 - 传说级
  换算：1极品灵石 = 100上品灵石 = 1,000,000下品灵石
  灵气含量：下品灵石的1,000,000倍
  颜色：七彩流光
  用途：化神期以上，战略资源
```

## 二、使用场景举例

### 2.1 日常生活
- 一个馒头：5-10灵萃
- 一碗灵米饭：20-30灵萃
- 普通客栈住宿（一天）：50-100灵萃
- 粗布衣服：200-300灵萃（2-3灵珠）

### 2.2 修士交易
- 低阶灵草：1-5灵珠（100-500灵萃）
- 低阶法器：10-50灵石（10,000-50,000灵萃）
- 炼气期丹药：5-20灵石
- 基础功法：50-200灵石

### 2.3 大型交易
- 洞府租赁（月）：100-500灵石
- 飞剑（低阶）：300-800灵石
- 筑基丹：5,000-10,000灵石（5-10中品灵石）
- 小型灵脉：100,000灵石起（100中品灵石）

## 三、金融操作换算

### 3.1 陆衍的启动资金修正
原设定：0.6下品灵石 → 修正：600灵萃 或 6灵珠
实际写作中统一使用灵萃为单位，避免小数

### 3.2 收益率计算
保持百分比不变，因为这是比例关系
例：33.3%收益率，无论用什么货币单位都一样

### 3.3 大额数字处理
- 10,000灵石以下：用灵石+灵珠表示（如：125灵石8灵珠）
- 10,000-100,000灵石：可用中品灵石表示（如：1中品灵石25灵石）
- 100,000以上：考虑用上品灵石

## 四、写作规范

### 4.1 数字表示
- 避免小数点：用灵萃代替小数部分
- 统一格式：`[数量][单位]`，如：`150灵萃`、`3灵珠`、`25灵石`
- 大额数字：适当使用中品/上品灵石简化

### 4.2 对话中的使用
- 普通人：多用灵萃、灵珠
- 低阶修士：灵石为主，偶尔用灵珠
- 高阶修士：中品灵石以上
- 陆衍：根据场景灵活使用，体现专业性

### 4.3 金融之眼显示
```
【个体：陆衍（炼气一层）】
【资产净值：-200灵萃（欠杂役房伙食费）】
【现金流：0】
【信用评级：垃圾级】
```

## 五、与现实的类比

为了读者理解，可在作者有话说中简单类比：
- 1灵萃 ≈ 1分钱人民币
- 1灵珠 ≈ 1角钱人民币  
- 1下品灵石 ≈ 1元钱人民币
- 1中品灵石 ≈ 100元钱人民币
- 1上品灵石 ≈ 10,000元钱人民币

但正文中不直接提及现实货币，保持世界观纯净。
