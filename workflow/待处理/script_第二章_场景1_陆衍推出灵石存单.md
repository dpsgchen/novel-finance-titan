# 剧本：第二章 场景1 - 陆衍推出灵石存单

## 基本信息
- **场景编号**：第二章 场景1
- **场景标题**：陆衍推出灵石存单
- **预计字数**：约2000字
- **前情提要**：陆衍通过做空战争赚到3块下品灵石（30万灵萃），开始规划金融帝国

## 时间地点
- **时间**：穿越后第三周，战争结束第五天
- **地点**：坊市边缘小院（陆衍新租的办公地点）
- **氛围**：创业初期，简陋但专业

## 人物
- **陆衍**：冷静理性，开始展现金融家思维
- **老陈**：第一个客户，忠实支持者
- **其他散修**：观望、怀疑、好奇

## 情节要点

### 1. 场景开场
- 陆衍租下坊市边缘小院，月租5000灵萃
- 简单装修：挂牌"陆氏资产管理"，门口贴公告板
- 金融之眼扫描：计算投入产出比，设定盈利目标

### 2. 产品设计
- **问题发现**：散修们战后资产闲置，不知如何管理
- **市场调研**：传统钱庄利率低（年化1-2%），服务差
- **产品创新**：设计"陆氏灵石存单"
  - 活期：年化5%，随时存取
  - 定期三个月：年化8%
  - 定期六个月：年化10%
- **风险控制**：准备金制度，分散投资，透明管理

### 3. 营销策略
- **公开透明**：门口贴出详细产品说明、利率计算
- **示范账户**：用自己3块下品灵石做示范，每日更新收益
- **口碑传播**：让老陈等早期客户帮忙宣传

### 4. 第一个客户
- 老陈带朋友来：战争期间看到陆衍的操作，信任建立
- 陆衍详细解释：产品原理、风险、收益
- 签契约：明确条款，按手印，用宗门印章具法律效力
- 第一笔存款：5000灵萃

### 5. 市场反应
- 坊市议论："这么高利率？骗人的吧？"
- 传统钱庄关注：万通钱庄掌柜派人暗中观察
- 散修观望：想存但怕风险

### 6. 陆衍的金融思维
- **核心理念**：金融的本质是信任和效率
- **竞争优势**：信息透明、专业管理、高收益
- **长期规划**：建立信用体系，发行可转让存单

### 7. 结尾钩子
- 三天后：存款总额达到50000灵萃
- 万通钱庄掌柜亲自来访："小友，你这生意...做得有点大啊。"
- 陆衍微笑："钱掌柜，这才刚刚开始。"

## 金融亮点
1. **产品设计**：差异化利率，满足不同需求
2. **风险管理**：准备金、分散投资、透明披露
3. **营销创新**：示范账户、公开透明、口碑传播
4. **信用建立**：契约法律效力，宗门印章背书

## 人物塑造
- **陆衍**：从交易员向金融家转型，开始建立体系
- **老陈**：从客户向"代言人"转变，体现信任传递
- **钱庄掌柜**：传统势力的警惕和试探

## 写作要求
1. 保持陆衍冷静理性、专业务实的性格
2. 金融产品描述要清晰易懂
3. 展现散修们的心理变化：怀疑→观望→尝试
4. 埋下传统钱庄冲突的伏笔
5. 字数：约2000字
6. 文风：专业但不晦涩，有创业激情
